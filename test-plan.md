# Test Plan for [Cafedaria](https://cafedaria.com/index.html) Website

## Scope
This test plan covers functional, UI and basic checkout flow testing for the cafedaria.com coffee shop website.  
Pages under test include Home, Menu, Catalog, Checkout and Contact pages.

## Test Types
- Manual Functional Testing
- UI / UX Testing
- Smoke Testing
- Regression Testing
- Cross-browser Testing
- Responsive Testing

## Environment
- Browsers: Chrome, Safari  
- Devices: Desktop, Mobile  
- OS: macOS, iOS

## Entry Criteria
- Website is accessible
- Test documentation is prepared
- Test environment is stable

## Exit Criteria
- All planned test cases are executed
- No critical defects remain open
- Test results are documented
